카메라에서 물체를 추적한 정보를 받아 로봇이 물체를 추종하게끔 움직여 주는 노드

기능 설명
1. object_position 토픽을 입력 받아 로봇을 움직여 물체를 추척한다.
    - object_position은 최신 값만 사용한다.
2. webrtc_unitree_ros.py를 이용해서 로봇을 움직인다.
3. 추적을 On/Off 할 수 있는 서비스가 있다.
4. 추적이 On 일 때 object_position 토픽의 정보를 기반으로 로봇을 움직이는 타이머 루틴이 시작된다.
5. 추적이 Off 일 때 타이머 루틴이 중지된다.
6. 최대 추적 속도를 지정할 수 있다.
7. 추적 절차
    1. X축으로 목표가 가운데 오도록 회전
    2. Y축으로 목표가 화면 하단에 위치하도록 전진
    3. 목표가 X축 중심에서 멀어지면 1번 부터 다시 시도
8. 파라미터
    - X축으로 허용 범위 : 0.01~0.5
    - Y축 하단 추적 멈춤 범위 : 0.01~0.5
    - 타이머 주기 : 0.01~1.0
    - 최대 추적 속도 : 0.01~1.0
